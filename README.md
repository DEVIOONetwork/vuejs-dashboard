# VueJS dashbaord template (api & client)

## Installation
```
git clone https://github.com/StudioDEVIOO/vuejs-dashboard.git
cd vuejs-dashboard
```
### Install dependencies
```
cd api
npm install
cd ..
```
```
cd dashboard
npm install
cd ..
```
